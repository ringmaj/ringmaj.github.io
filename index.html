    Recursive Tree Analysis Tool

    A personal project I am working on to simplify the analysis of recursive trees by the recursion tree method using OpenGl/C++. This tool allows users to follow a simple graphical user interface to input a recursive equation such as T(n) = 4T(n/2) + 2n2 and view the output as a detailed recursive tree. Users are also able to view other details such as depth, size of sub-problem, number of nodes, workload per node, and total workload per depth. In this project, only OpenGl primitives were used.

    Parking Analytics Database

    For my CSE-111 Database systems project, the primary goal for our web application is to develop a program where users can view a graphical layout of the campus parking lot and see the amount of spots available through highlighted sections. The user can also select a spot and view the current occupant info (name, age, sex, parking duration) and also modify and search for specific spots/areas.

    UAV Team

    Through UC Merced's Engineering Service Learning program, I worked with interdisciplinary teams to develop a program which utilizes a drone to autonomously fly through a route and take images which are later processed into our computer program and is able to detect pierce’s disease in crops.I developed the user interface for our program in java and also wrote code to analyze areas of concern through NDVI.

    Skateboarding Physics Simulator

    For my course, CSE 170 - Introduction to Computer Graphics, I developed a program to simulate and analyze skateboarding tricks in 3D using OpenGL. I used a custom graphs/Bezier class to display and control rotations in all axis using OpenGL/C++.

    GRE Vocab Builder

    A personal project where I am developing an android application to help aspiring students study for their GRE exam. This is a quiz/card based system where users can save scores, prioritize words for better learning, and also review overall progress and highlight areas of improvement.

    RNA Sequencing Analysis Web Pipeline

    With my organization, the Quantitative Project, I worked on developing a web pipeline for for laboratory use at UC Merced. This web pipeline will allow users to login and access their RNA-seq data and perform analysis on the web after customizing and selecting their workflow. All data will be stored and available for easy access and is stored securely. This project is still in progress and utilizes PHP, HTML,CSS, Python, and SQL.
